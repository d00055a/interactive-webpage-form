# Interactive Webpage Form

## Overview

This project is an interactive web form application built with HTML, CSS and Vanilla JavaScript. It was developed independently as part of the initial tasks of a remote Web Development Internship at Codveda Technologies.

The application focuses on client-side form handling, validation logic and basic user interaction without using any frontend frameworks or backend services.

## Project Scope

This is a frontend-only learning project aimed at practicing JavaScript fundamentals related to form validation and DOM manipulation.

## Features

- Interactive form with multiple input types:
  - Full name
  - Email
  - Password
  - Age
  - City (dropdown)
  - Gender (radio buttons)
  - About (textarea)
- Client-side validation using JavaScript:
  - Required field checks
  - Email format validation
  - Password length validation
  - Age range validation
- User feedback via alert on successful submission
- Clean and responsive layout built with CSS

## Tech Stack

- HTML5 – Page structure
- CSS3 – Styling and responsive layout
- JavaScript (Vanilla) – Form validation logic and event handling
